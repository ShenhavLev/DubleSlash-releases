# DubleSlash — Releases

Public download artifacts for the [DubleSlash](https://github.com/ShenhavLev/DubleSlash-method) desktop app.

The app's source code lives in a private repo. This repo only holds the signed `.dmg` installers and the Tauri updater manifest (`latest.json`) so first users and the auto-updater can fetch them without GitHub auth.

## Install (macOS, Apple Silicon)

Grab the latest `.dmg` from the [Releases page](https://github.com/ShenhavLev/DubleSlash-releases/releases/latest).

Requires macOS 13+ on an M1 (or newer) Mac.
